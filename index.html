About Me

I am a software developer. I was a chef with an associate's degree in Culinary Arts and Business. I'm fascinated by the way data is changing the food industry. About 9 months ago a couple of chefs asked if I could develop a program that would help improve the processes of kitchen management. I without hesitation told them I thought I could make that happen. I proceeded to build out the program in Excel. Fast forward to today, I am now heading a team of 3 student developers as we build out a full stack kitchen management application. I am skilled in Vue.js, Cascading Style Sheets (CSS), Node.js, HTML, and JavaScript. I am now a software developer with a Certificate focused on Full Stack Software Development from Boise CodeWorks.

Stach

I created MuStach as my final project for Boise Codeworks. It took around 32 hours to build. One of my favorite projects thus far. It has C# backend using the dotnet framework, and a MySQL database. The front end is in Javascript with the Vue.js framework. I used a testing tool built by our class TA. Once the routes passed the test the rest of the project was super easy. I wanted to make the app interactive, so add draggability to the posts.

Fidget

Fidget is based on the Kanban methodology. Which allows for you to create boards for projects on which you can create lists of tasks that need to be completed. It has the option of moving tasks between lists and adding comments to tasks. This was my first full stack project I completed with a group of the two other student developers. It is written with a Javascript front end in the Vue.js framework. The back end used the Node.js framework also written with Javascript. We utilized the Mongoose ORM to input data into the MongoDB database.
